<br/>

## If using public repo

Push your code to the repo using

```
git add .
git commit -m "something"
git push origin main
```

then netlify will automatically deploy the website.


## If using private repo

`npm start` to run the website in the development mode. -> http://localhost:3000 

run `npm run build` and upload 'build' directory on netlify for actual hosting.
